# Project Overview

Stock market forecasting is one of the most challenging problems in financial analytics because stock prices are highly dynamic and sequential in nature.

This project focuses on predicting future stock closing prices of Apple Inc. (AAPL) using:

- Linear Regression
- Random Forest
- LSTM (Long Short-Term Memory)

The project demonstrates the complete machine learning workflow from:

✔ Environment Setup  
✔ API-based Data Collection  
✔ Data Cleaning  
✔ Feature Engineering  
✔ Exploratory Data Analysis  
✔ Regression Modeling  
✔ Deep Learning Forecasting  
✔ Model Evaluation  
✔ Future Price Forecasting  

---

# 🎯 Project Objectives

The main objectives of this project are:

- Fetch real-time stock market data using Yahoo Finance API
- Perform preprocessing and feature engineering
- Analyze historical stock behavior
- Train machine learning models
- Forecast future stock prices
- Compare traditional ML models with Deep Learning models
- Visualize actual vs predicted stock prices
